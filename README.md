# 🚀 Yuri Loureiro | Creative Full Stack Developer

Este é o meu portfólio profissional de 2026. Um projeto focado em **performance**, **experiência do usuário (UX)** e **micro-interações fluidas**, desenvolvido para apresentar meus projetos e habilidades técnicas de forma minimalista e impactante.

---

## ✨ Destaques Técnicos

- 🌍 **i18n (Internationalization):** Sistema de tradução dinâmica (PT/EN) via React Context, permitindo troca de idioma instantânea sem recarregar a página.
- 🖱️ **Micro-interações Premium:**
  - Cursor customizado reativo.
  - Efeito magnético em botões e cards (Framer Motion).
  - Animações de texto "Reveal" letra por letra.
- 📧 **Sistema de Contato Real-time:** Integração com a API do **Resend** para envio de e-mails autenticados via domínio próprio.
- 📱 **Responsividade Total:** Design adaptável para todos os dispositivos, priorizando a legibilidade e o toque.
- ⚡ **Performance:** Otimização de imagens (Next/Image) e fontes (Next/Font) para carregamento ultra-rápido.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js 15 (App Router)]
- **Linguagem:** [TypeScript]
- **Estilização:** [Tailwind CSS]
- **Animações:** [Framer Motion]
- **E-mail:** [Resend SDK]
- **Ícones:** [Lucide React]
- **Deploy:** [Vercel]
